# Statistics Module

The Statistics module provides detailed business insights through four interactive Chart.js visualizations, KPI cards, and a product performance table.

## Overview

Access comprehensive statistics at `/estadisticas` with:

<CardGroup cols={2}>

  </Card>
  <Card title="Sales Trends" icon="chart-line">
    Monthly sales progression with line chart
  </Card>
  <Card title="User Demographics" icon="users">
    Gender distribution with doughnut chart
  </Card>
  <Card title="Product Performance" icon="box">
    Top selling products with horizontal bar chart
  </Card>
</CardGroup>

## Route Configuration

The statistics page is defined in `routes/web.php`:

```php
Route::get('/estadisticas', fn() => view('estadisticas'))->name('estadisticas');
```

<Note>
  View file: `resources/views/estadisticas.blade.php`
</Note>

## Chart Visualizations

The statistics module includes four distinct Chart.js visualizations:

### 1. Monthly Sales Chart (Line Chart)

Tracks sales progression over six months:

```html
<div class="col-md-6">
    <div class="card">
        <div class="card-header">
            <h5>Ventas Mensuales</h5>
        </div>
        <div class="card-body">
            <canvas id="ventasChart" height="200"></canvas>
        </div>
    </div>
</div>
```

#### Chart Configuration

```javascript
new Chart(document.getElementById('ventasChart').getContext('2d'), {
    type: 'line',
    data: {
        labels: ['Enero','Febrero','Marzo','Abril','Mayo','Junio'],
        datasets: [{
            label: 'Ventas ($)',
            data: [5000, 7500, 6800, 9200, 11500, 14200],
            borderColor: 'rgb(75,192,192)',
            backgroundColor: 'rgba(75,192,192,0.1)',
            tension: 0.3,
            fill: true
        }]
    },
    options: {
        responsive: true
    }
});
```

## Chart Types Overview

<CardGroup cols={2}>
  <Card title="Line Chart" icon="chart-line">
    **Best for**: Time series data, trends over time
    
    **Used for**: Monthly sales progression
  </Card>
  <Card title="Doughnut Chart" icon="circle-notch">
    **Best for**: Part-to-whole relationships with center hole
    
    **Used for**: Gender demographics
  </Card>
  <Card title="Bar Chart" icon="chart-bar">
    **Best for**: Comparing quantities across categories
    
    **Used for**: Product sales comparison (horizontal)
  </Card>
  <Card title="Pie Chart" icon="chart-pie">
    **Best for**: Showing proportions of a whole
    
    **Used for**: Revenue distribution by category
  </Card>
</CardGroup>

## Customization Guide

<Steps>
  <Step title="Update Chart Data">
    Modify the `data` arrays in each chart configuration:
    
    ```javascript
    data: [5000, 7500, 6800, 9200, 11500, 14200]  // Replace with your data
    ```
  </Step>

  <Step title="Change Colors">
    Update the `backgroundColor` and `borderColor` properties:
    
    ```javascript
    backgroundColor: 'rgba(255,99,132,0.8)'  // New color
    ```
  </Step>

  <Step title="Adjust Chart Options">
    Customize chart behavior in the `options` object:
    
    ```javascript
    options: {
        responsive: true,
        maintainAspectRatio: false,
        plugins: {
            legend: { position: 'top' }
        }
    }
    ```
  </Step>
</Steps>

<Warning>
  Ensure Chart.js is loaded before initializing charts. Include the library in your layout:
  
  ```html
  <script src="https://cdn.jsdelivr.net/npm/chart.js"></script>
  ```
</Warning>
